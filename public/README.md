# Public Assets

이 폴더의 파일들은 향후 실제 자산으로 교체해야 합니다.

## 필수 교체 항목

### `og-default.jpg` (필수)
- 사이즈: **1200 × 630px**
- 용도: SNS 공유 시 표시되는 대표 이미지 (Open Graph)
- 톤: 다크 베이스 + 골드 액센트의 시네마틱 비주얼 권장
- 임시로는 logo.svg를 PNG로 변환해 사용 가능

### `logo.svg` (현재 placeholder)
- 정식 BI/CI가 확정되면 교체
- SVG 우선, PNG 보조

### `favicon.ico`, `favicon.svg`, `apple-touch-icon.png`
- favicon.svg는 작성됨
- 추가로 favicon.ico (32x32) 와 apple-touch-icon.png (180x180) 권장
- https://realfavicongenerator.net 에서 생성 가능

### 폰트 (선택)
사이트는 Google Fonts CDN을 사용 중입니다. 자체 호스팅을 원하면 `fonts/` 폴더에 추가하세요.
